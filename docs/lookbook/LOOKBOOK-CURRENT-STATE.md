## NEXT STEP: 2-PAGE SPREAD STRATEGY

The rebuild should use a **2-page spread approach** — every left+right page pair works as one visual/content unit with a single purpose. This forces discipline:

- If content doesn't earn a full spread, it gets merged or cut
